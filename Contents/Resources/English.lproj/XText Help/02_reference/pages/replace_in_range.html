replace_in_range

replace_in_range(s_index, e_index, new_text)

Replacing sub-text in specefied range.

Parameters
  • s_index (integer) : an index of the beginning of the range
  • e_index (integer) : an index of the ending of the range
  • new_text : (Unicode text, string or XText) : a new text should be placed in the range.
Result

XText : a new XText instance
