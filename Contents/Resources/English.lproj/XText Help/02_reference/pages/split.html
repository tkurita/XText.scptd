split

split(a_text, a_delimiter)

Make a list with splitting a text with specified delimiter.

The method should be called between store_delimiters() and restore_delimiters()If the class of a_delimiter is Unicode text, the class of a_text must be Unicode text. The classes of elements of a retuend list are same to the class of a_text.

Parameters
  • a_text (string, Unicode text or XText) : a text to be split into a list
  • a_delimiter (string, Unicode text) : a delimiter used to split a_text
Result

list of string or Unicode text
