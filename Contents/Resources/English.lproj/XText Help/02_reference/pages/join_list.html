join_list

join_list(a_list, a_delimiter)

Join a list of text with specified delimiter into a Unicode text.

The method should be called between store_delimiters() and restore_delimiters()

Parameters
  • a_list (list) : a list consisting of strings or Unicode texts
  • a_delimiter (string or Unicode text) : a delimiter text to join a_list
Result

Unicode text
