sprintf

sprintf(format_text, param_list)

Generate formatted text using printf command.

Parameters
  • format_text (text or XText) : format
  • param_list (list or XList) : a list ot texts to be inserted.
Result

text

Example
use XText : script "XText"
XText's sprintf("%s : %3.2e", {"Value", 10})
-- result : "Value : 1.00e+01"
