join_list

join_list(a_list, a_delimiter)

string もしくは Unicode text のリストを、指定された区切り文字を使って連結した文字列（Unicode text）を作ります。

store_delimiters() と restore_delimiters() の間で実行してください。

Parameters
  • a_list (list) : string もしくは Unicode text を要素としたリスト
  • a_delimiter (string or Unicode text) : リストの接続に使う区切り文字
Result

Unicode text
