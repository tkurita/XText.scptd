replace

replace for a_text from old_text by new_text

文字列の中に含まれる特定の文字列を別の文字列に置換します。結果は Unicode text class になります。

store_delimiters() と restore_delimiters() の間で実行してください。

Parameters
  • a_text (string, Unicode text or XText) : 置換処理を行いたい文字列
  • old_text (string or Unicode text or XText) : 置き換えたい文字列
  • new_text (string, Unicode text or XText) : 置き換える文字列
Result

Unicode text : a_text に含まれる old_text を new_text に置き換えた文字列
