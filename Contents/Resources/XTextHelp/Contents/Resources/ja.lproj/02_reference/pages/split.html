split

split(a_text, a_delimiter)

文字列を指定した delimiter で分解した文字列のリストを取得します。

store_delimiters() と restore_delimiters() の間で実行してください。a_delimiter が Unicode text の場合は、a_text も Unicode text である必要があります。また返り値のリストの要素の class は a_text の class と同じになります。

Parameters
  • a_text (string, Unicode text or XText) : リストに分解したい文字列
  • a_delimiter (string, Unicode text) : リストに分解する際の区切り文字
Result

list of string or Unicode text
