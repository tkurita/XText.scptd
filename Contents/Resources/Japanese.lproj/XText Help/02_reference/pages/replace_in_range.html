replace_in_range

replace_in_range(s_index, e_index, new_text)

指定した範囲のテキストを置き換えます。

Parameters
  • s_index (integer) : 置換範囲の開始位置
  • e_index (integer) : 置換範囲の終了位置
  • new_text : (Unicode text, string or XText) : 置換テキスト
Result

XText : 指定した範囲を new_text で置き換えた新しい XText のインスタンス
